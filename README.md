**Image Color Classification using Machine Learning**

A simple machine learning project that classifies images based on their dominant RGB color values using Logistic Regression.

Overview

This project demonstrates a basic image classification pipeline where images are processed to extract RGB features, and a machine learning model is trained to predict color labels.

It is designed as a beginner-friendly project to understand:

Image preprocessing
Feature extraction (RGB values)
Supervised learning
Model evaluation
Features
Load images from dataset
Extract RGB color features
Encode labels using LabelEncoder
Train-test data splitting
Train a Logistic Regression model
Evaluate model accuracy
Visualize sample images
Model Used
Logistic Regression (from scikit-learn)

The model learns patterns from RGB values to classify images into predefined color categories.
